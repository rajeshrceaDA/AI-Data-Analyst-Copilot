# AI Data Analyst Copilot

An AI-powered career assistant designed to help aspiring Data Analysts create ATS-friendly resumes, analyze job descriptions, and optimize resumes using AI.

## Features

- AI Professional Summary Improvement
- ATS Resume Checker
- Resume Generator (DOCX)
- Profile Management
- Resume Preview
- OpenRouter AI Integration
- Multiple Projects & Experience Support
- Categorized Technical Skills

## Tech Stack

- Python
- Streamlit
- OpenRouter AI
- JSON
- python-docx
- Regex
